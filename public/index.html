Live Streaming

Broadcast simultaneously to YouTube and Facebook

Stream Setup

How it works

  1. 1 Select platforms and click Go Live — we create the stream sessions instantly.
  2. 2 Copy the RTMP URL and Stream Key for each platform.
  3. 3 Open OBS Studio, go to Settings → Stream, and add each platform as a custom RTMP service.
  4. 4 Use multiple scene collections or a tool like Restream to broadcast to all at once.
  5. 5 When done, click End Stream to close the broadcast on all platforms.
Note: TikTok and Instagram Live require their native apps and don't offer public APIs for external streaming tools.
